## üöÄ How to Use the Tool

1. **Create your own copy of the template**  
   ‚Üí *File ‚Üí Make a Copy ‚Üí Save to your Google Drive.*  
   (The original file is view-only.)

2. **Install a GPT extension for Google Sheets**  
   Supported: *DocGPT*, *AiAssistWorks*, *GPT for Sheets*, and similar tools.

3. **Connect your personal API key**  
   Configuration is done directly via the chosen extension‚Äôs interface.

4. **Activate GPT formulas**  
   Remove the `/` symbol before `=` in pink-highlighted cells (e.g., `/=GPT(...)` ‚Üí `=GPT(...)`).

5. **Enter your market parameters**  
   Define market name, geography, period, format, and sources ‚Äî the system will automatically generate a report.

6. **Save the generated results (optional)**  
   Copy the generated text and paste it as *values only* to stop automatic updates and avoid extra token usage.
